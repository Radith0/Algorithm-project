# PathFinder - BFS Algorithm for Maze Solving
A Java program using Breadth-First Search (BFS) to find the shortest path in a maze represented by a 2D array.

Features
Represents the maze with a 2D array
Uses BFS to find the shortest path
Performance analysis included
Setup and Run
Clone the repository
Navigate to the project directory
Compile the Java program
Run with a maze input file
Performance
Time complexity: O(V), where V is the number of cells
Empirical data shows linear growth with input size
